# SDL2 Platformer 2D Engine

Demo project to learn writing a 2D Game Engine, C++ / [SDL2](https://www.libsdl.org/) programming.

## How to install

On Ubuntu / Debian systems, run the `indstall.sh` to install runtime dependencies.
